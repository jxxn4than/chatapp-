# Chat App - Sitio Web (v2)

Versión con sistema de registro tipo TikTok (teléfono/email + OTP demo).

## Instalación

```bash
npm install
npm run dev
```

Para usar el servidor Socket.IO (opcional):

```bash
node server.js
```

El código incluye un OTP demo (código 1234) para iniciar sesión rápido.
